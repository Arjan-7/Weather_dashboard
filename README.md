# Weather Dashboard 🌤️

A simple Streamlit app that shows current weather for any city using the OpenWeatherMap API.

## Features

- Live temperature, feels-like, humidity, clouds, and wind.
- City text input, clean metric cards.
- Raw JSON and dataframe for learning.

## Run Locally

```bash
git clone repo link
cd your repo name
pip install -r requirements.txt
streamlit run app.py
```

Add your OpenWeatherMap API key in `app.py` before running.
